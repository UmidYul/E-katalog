# EK.UA: функциональный аудит

Дата аудита: 23 февраля 2026  
Домен: `https://ek.ua/`

## 1) Важная оговорка по "всем страницам"

`ek.ua` содержит очень большое количество товарных и служебных URL (десятки тысяч+).  
Поэтому ниже зафиксирован **полный функционал по всем основным типам страниц и шаблонам** (листинги, карточки, цены, рейтинги, статьи, бренды, B2B), а не ручной просмотр каждого SKU-URL по отдельности.

## 2) Покрытые типы страниц

- Главная: `https://ek.ua/en/en/`
- Категорийный листинг: `https://ek.ua/en/ek-list.php?katalog_=122&order_=price&years_=1`
- Карточка товара (PDP): `https://ek.ua/en/NOMI-I2320.htm`
- Страница цен модели: `https://ek.ua/en/prices/nomi-i2320/`
- Отзывы модели: `https://ek.ua/en/ek-item.php?resolved_name_=NOMI-I2320&view_=review`
- Раздел обратной связи по категории: `https://ek.ua/en/ek-forum.php?katalog_=122`
- Рейтинг категории: `https://ek.ua/en/z122.htm`
- Популярные запросы: `https://ek.ua/en/e122.htm`
- Справочник/описания категории: `https://ek.ua/en/wiz_char.php?id=122`
- Каталог брендов: `https://ek.ua/en/katalog/`
- Контент (список статей): `https://ek.ua/en/ek-post.php?ida_=0&katalog_=1&mode_=short`
- Контент (страница статьи): `https://ek.ua/en/post/5122/122-why-do-smartphones-start-to-slow-down-and-what-amount-of-ram-is-now-considered-optimal/`
- B2B/для магазинов: `https://ek.ua/en/for-shops`
- B2B docs: `https://account.ek.ua/shop/docs/how-work-nadavi?lang=en`, `.../connect-store?lang=en`, `.../tariffication?lang=en`

## 3) Глобальный функционал сайта

- Поиск по сайту (глобальная строка поиска).
- Выбор страны/региона.
- Переключение языков (UA/EN/RU).
- Кнопка/доступ к авторизации (`Log in`).
- Мега-меню категорий с подкатегориями.
- Быстрые популярные категории в шапке.
- Футер с:
  - контактами/служебными ссылками;
  - privacy policy;
  - мультистрановыми зеркалами (UK/USA/PL/KZ);
  - переключением dark mode;
  - блоком Compare / Comparison / Bookmarks / History.
- Быстрая отправка ошибки контента (`Ctrl+Enter` + "Report an error in the description").

## 4) Функционал каталога (листинги категорий)

- Хлебные крошки.
- Заголовок категории и счетчик моделей (`prices on N models`).
- Технические "preset" фильтры (`new`, `clear` и т.д.).
- `Save List` (сохранение выборки).
- Сортировки:
  - по рейтингу;
  - по цене (asc/desc);
  - по отзывам;
  - по дате добавления;
  - по техпараметрам (экран, RAM, память, батарея, вес и др.).
- Блок "more parameters" с большим количеством параметров.
- Фильтр "with delivery in Ukraine".
- В товарной карточке листинга:
  - мини-превью товара;
  - техническое summary;
  - теги/лейблы (New, NFC, 120Hz, waterproof и т.д.);
  - вариации модели (например объем памяти);
  - цветовые варианты;
  - счетчики контента (`User reviews`, `Discussion`, `Media reviews`, `Photos`, `Videos`, `Manuals`, `Prices`);
  - кнопка/действие `compare`;
  - прямой переход "Compare prices and buy ...";
  - инлайн вывод магазинов и цен;
  - счетчик `X stores / Y offers`;
  - ленивый догруз списка магазинов (`Loading online stores list.`).
- Блок перекрестных рекомендаций "You may be interested in".

## 5) Функционал карточки товара (PDP)

- Табы карточки:
  - Prices;
  - Description;
  - Specifications;
  - User reviews;
  - Ask Question;
  - Accessories.
- Фото-галерея модели.
- Сводный ценовой диапазон (from ... up to ...).
- "How to buy" + список магазинов с ценами.
- Быстрые пользовательские действия:
  - `add to list / my lists`;
  - `add to comparison`.
- Блок короткой сводки характеристик.
- Полная таблица спецификаций по секциям:
  - Display;
  - Hardware;
  - Camera;
  - Connectivity;
  - Features;
  - Power;
  - General.
- Технические и юридические дисклеймеры по описанию.
- `Report description mistake` (репорт ошибок в описании).
- Встроенные редакционные ссылки/материалы по теме.
- Полный блок "Buy [model]" с офферами:
  - название товара у магазина;
  - стаж магазина ("with us X years");
  - город;
  - гарантия;
  - кнопка `Report`;
  - CTA `Buy in shop`.
- Блок аксессуаров к модели (кросс-сейл).

## 6) Функционал страницы цен модели (`/prices/...`)

- Отдельная страница агрегатора цен по конкретной модели.
- Сортировка офферов (как минимум `by rating`, `by price`).
- Детальный список предложений по магазинам.
- Метаданные магазина в оффере:
  - возраст/стаж;
  - город;
  - гарантия;
  - репорт;
  - кнопка перехода в магазин.

## 7) Отзывы, Q&A, обсуждения

- Вкладка `User reviews` в карточке модели.
- Состояние "No reviews yet".
- CTA `Write Review`.
- Блок `Ask Question` (присутствует в карточке модели).
- Категорийный форум улучшений (`ek-forum.php?katalog_=...`):
  - форма предложений/фидбека;
  - требование логина для публикации;
  - древовидные ответы (`Answer`);
  - публичные ответы модераторов;
  - история предложений по фильтрам/параметрам.

## 8) Рейтинги и аналитические разделы

- Страница рейтинга (`z*.htm`):
  - `Top 10` категории;
  - позиция, доля популярности, диапазон цены;
  - быстрые ссылки для каждой позиции:
    - `User reviews`;
    - `Photo/Video`;
    - `Where to buy?`.
- Объяснение методики рейтинга.
- Диаграммы/срезы по параметрам (бренды, экран, CPU, батарея и др.).
- Обсуждение на странице рейтинга + пагинация (`next 20`).
- Фильтры на странице рейтинга (цена, бренды и мн. др.).

## 9) Раздел "Popular requests"

- Список популярных запросов/интентов внутри категории.
- Рекомендованные "фичи" для подбора.
- Полноценная панель фильтров (аналог больших листингов):
  - брендовые фильтры;
  - серии;
  - сценарные пресеты (gaming, camera phone, seniors и т.п.);
  - технические фильтры по группам (экран/камера/CPU/батарея/связь и т.д.).

## 10) Раздел "Descriptions" (`wiz_char.php`)

- Энциклопедический/справочный каталог моделей по брендам.
- Ссылочный список моделей каждого бренда.
- Блок `Brand rating`.
- `All brands / Main brands`.
- Серии.
- Набор фильтров/характеристик по группам:
  - Features;
  - Display;
  - Camera;
  - CPU/RAM/Storage;
  - Connectivity;
  - Battery/Charging;
  - Materials/Weight/Dimensions.
- Ссылка "Questions and requests regarding product selection (search)".

## 11) Бренды и брендовые каталоги

- Общий `Brands Catalog` по алфавиту (A-Z, цифры).
- Переход к бренд-страницам.
- На бренд-странице:
  - число моделей;
  - число отзывов;
  - число обзоров;
  - число офферов магазинов;
  - набор категорий бренда;
  - списки моделей по категориям.

## 12) Контент/медиа функционал

- Лента редакционных материалов (обзоры/подборки/советы).
- Карточки статей с:
  - датой;
  - автором;
  - превью текста.
- Полная страница статьи:
  - автор;
  - роль автора;
  - время чтения;
  - дата обновления;
  - структурированные секции статьи.
- Встроенные товарные блоки внутри статей (контекстная коммерция):
  - товар;
  - цена;
  - переход к модели.

## 13) B2B функционал для магазинов (`for-shops` + account docs)

- Продуктовая лендинг-страница для eShop партнеров.
- Регистрация магазина / подключение (`Add a shop`, `Connect a store`).
- Форма лидогенерации для подключения:
  - "How did you know about us?"
  - отправка заявки.
- Описание механики загрузки прайс-листов (manual/auto).
- Аукционная модель выдачи предложений (bid-based placement).
- Тарификация за уникальные переходы.
- Документация по:
  - how system works;
  - pricing/tariffication;
  - размещению магазина;
  - banner ads;
  - brand promo;
  - terms/public offer/contacts.
- Личный кабинет для магазинов (`account.ek.ua`) — закрытая зона.
- Реалтайм статистика переходов (описана в документации).

## 14) Функции персонализации и пользовательских списков

- Механика сравнения товаров (`add to comparison` + глобальный блок compare).
- Закладки (`Bookmarks`).
- История (`History`).
- Списки пользователя (`my lists`, `Save List`).
- Региональные/языковые настройки.
- Темная тема (`Dark mode`).

## 15) Дополнительные наблюдения

- На части страниц есть отдельные CTA `Report` по каждому офферу магазина.
- Есть отдельные хабы "rating", "popular requests", "descriptions", которые работают как самостоятельные сценарии выбора товара.
- Есть интеграция с внешними региональными проектами (pricehub/e-catalog/e-katalog для других стран).

## 16) Что не удалось проверить полностью

- Закрытые разделы `account.ek.ua` (часть страниц требует авторизацию и/или возвращает `401 Unauthorized` для анонимного доступа).
- Прямые URL внутренних экранов "Comparison/Bookmarks/History" без добавленных товаров (интерфейс привязан к состоянию выбранных моделей).
- Невозможно вручную просмотреть каждую SKU-страницу из-за масштаба каталога; зафиксирован функционал на уровне шаблонов и основных сценариев.

# E-Katalog (Express MVP)

MVP-сайт агрегатора смартфонов по мотивам `deep-research-report.md`:

- независимый каталог устройств (бренд, модель, характеристики, варианты);
- слой офферов магазинов (цена, наличие, доставка, переход);
- поиск и фильтры каталога;
- сравнение устройств (лимит 4 модели для MVP);
- карточка устройства с блоком «Где купить» и мини-историей цен.

## Запуск

```bash
npm install
npm start
```

Сервис поднимется на `http://localhost:3000`.

## Основные страницы

- `/` — главная
- `/catalog` — каталог с фильтрами
- `/search` — поиск
- `/phone/:slug` — карточка модели
- `/compare?items=slug1,slug2` — сравнение
- `/about` — описание MVP

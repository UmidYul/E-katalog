# E-katalog Platform

- Backend services: `services/`
- Shared modules: `shared/`
- Infra and deployment: `infra/`
- Alembic migrations: `migrations/`
- Ops scripts: `scripts/`
- Frontend app: `frontend/`

## Frontend

```bash
cd frontend
cp .env.example .env.local
npm install
npm run dev
```

See `frontend/README.md` for full setup.

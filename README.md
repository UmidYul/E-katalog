# E-katalog UZ (Demo)

Прототип фронтенда под рынок Узбекистана c фейк-данными и минимальным backend на Express.js.

## Что реализовано
- Категория MVP: **Смартфоны**
- Интерфейс RU + UZ (переключение в каталоге)
- Каталог с фильтрами (поиск + бренд)
- Карточка товара с предложениями магазинов и отзывами
- Сравнение до 3 товаров
- Страница магазинов
- Регистрация (демо API)

## Запуск
```bash
npm install
npm start
```

Открой: `http://localhost:3000`
